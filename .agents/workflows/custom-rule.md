# AIアシスタントへの指示・ルール

ここに、私（AIアシスタント）に守ってほしいプロジェクト固有のルールや、毎回決まった手順で行うタスク（ワークフロー）を記述してください。

## ルールの記述例
- 新しいファイルを作成するときは必ず特定のディレクトリ（例: `src/`）に配置する。
- Gulpを用いたビルドタスクを実行してから作業を完了する。
- 丁寧な日本語でコメントを残す。

## ワークフローの記述例
具体的なタスクの手順を番号付きリストで記述すると、私がその通りに上から順番に実行します。

1. 〇〇のディレクトリに移動する。
2. 自動実行コマンド:
// turbo
3. `npx gulp` を実行する。

> **Tips:**
> コマンドの確認をスキップして自動的に実行させたいステップがある場合は、その直前に `// turbo` と記述してください。
> すべてのコマンドを自動実行したい場合は、このファイルのどこかに `// turbo-all` と記述します。
